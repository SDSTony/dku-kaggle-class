RMSE 손실함수 정의¶

rmse = lambda y, p: np.sqrt(np.mean((y - p) ** 2))

NumPy를 이용한 선형회귀¶

alpha = .001
theta0 = .1
theta1 = .1
epoch = 4

x = X.flatten()
print(x.shape)

(200004,)

rmses = []
for i in tqdm(range(epoch)):
    for x_i, y_i in zip(x, y):
        p_i =  theta0 + theta1 * x_i
        theta0 = theta0 + alpha * (y_i - p_i)
        theta1 = theta1 + alpha * (y_i - p_i) * x_i
    rmses.append(rmse(y, theta0 + theta1 * x))
